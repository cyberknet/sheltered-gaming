## About Our Minecraft Bedrock Server

### Platform Compatibility

Our Bedrock server supports:
- 🖥️ **Windows 10/11** (Bedrock Edition)
- 🎮 **Xbox Series X/S** and **Xbox One**
- 📱 **iOS** (via Game Pass app)
- 🤖 **Android** (via Game Pass app)
- 🎮 **Nintendo Switch**
- 🎮 **PlayStation 4/5** (via Game Pass)

### Getting Started

#### Windows/Console Players
1. **Own Minecraft Bedrock Edition** or have **Xbox Game Pass**
2. **Launch Minecraft** on your device
3. **Go to Friends tab** → **Play on servers**
4. **Add our server** using the IP below
5. **Join and play!**

#### Mobile Players (iOS/Android)
1. **Install Xbox Game Pass app** (or own Minecraft Bedrock)
2. **Look for our server** in the multiplayer section
3. **Connect** using our server IP
4. **Start your adventure!**
