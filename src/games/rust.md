## About Our Rust Server

Welcome to Sheltered Gaming's Rust server! This is where hardcore survival meets PvE collaboration. Whether you're a veteran or new to Rust, you'll find challenges and opportunities for epic gameplay.

### Server Rules

- **We only allow Solo and Duo Clans**. You are allowed at most one Alliance.
- **Only raid with your clan and/or alliance members**. If you raid with anyone else, it will be treated as a team size violation.
- **You can't claim monuments or default quarries.** Don't try to wall them off or otherwise permanently deny other players access.
- **Don't Be Awful**, keep your steam username, chat, VC, discord name, and in-game signs clean. Remember rule #1. Don't be awful.
- **We don't want cheaters**. If we think you have cheated, you'll get a permanent ban.
- **Please don't advertise other servers or websites**. We put a lot of work into this server, we're just asking you to be respectful.
- **All admin decisions are final**. We do our best to be fair.
- **If you suspect someone of cheating or a rule violation, report it via F7**. This goes to the server admins and also to Facepunch.

### Getting Started

1. **Install Rust** on Steam if you haven't already
2. **Copy the server IP** from the Server IP card below
3. **Open Rust** and go to the Server Browser
4. **Search for our server** or paste the IP directly
5. **Join and survive!**

NOTE: Due to anti offline raid protection, duo teams are required to create a clan. If you don't, your raid protection may not work.

### Server Features

- **Regular Wipes** - We wipe monthly during force wipe.
- **Active Admin Team** - We're here on Rust and Discord to help and maintain fair play
- **Community Events** - We have server event plugins to make PvE life more fun
- **Discord Integration** - Stay updated with in-game announcements on Discord

### Map Information

Our server uses a random map each month, and we try to include all monuments, biomes, and train lines. If you have a particular seed you'd like considered, send it to an admin on Discord.
