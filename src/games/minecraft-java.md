## About Our Minecraft Java Server

Welcome to Sheltered Gaming's Minecraft Java Edition server! Whether you're looking to build incredible structures, explore vast landscapes, or join a thriving survival community, this is the place for you.

### Server Specifications

- **Edition** - Minecraft Java Edition
- **Game Mode** - Survival with community collaboration
- **Difficulty** - Hard (respawn anchors recommended!)
- **Plugins** - Essential quality-of-life improvements (no pay-to-win)

### Getting Started

1. **Own Minecraft Java Edition** - If you don't have it, purchase from Minecraft.net
2. **Launch the Launcher** - Open the official Minecraft Launcher
3. **Add Server** - Click "Add Server" and enter our IP below
4. **Select Survival** - Choose Survival mode and join!

### Server Features

- **Vanilla+ Experience** - Vanilla Minecraft with helpful quality-of-life plugins
- **Grief Prevention** - Claim your land to protect your creations
- **economy System** - Trade with other players and earn currency
- **Regular Backups** - Your builds are safe and secure
- **Community Events** - Regular building contests and adventures
- **Shops & Markets** - Community trading hub for buying and selling

### Building Guidelines

- **Respect others' claims** - Don't build in or near claimed areas without permission
- **Clean up** - Remove temporary structures and farms
- **Lag-friendly builds** - Avoid excessive redstone or entity spam
- **Report griefing** - Use `/report` to let admins know about issues

### Progression Tips

1. **Early game** - Gather wood, stone, and find a safe spot to build
2. **Mid game** - Build a base, establish a farm, earn currency
3. **Late game** - Join collaborative projects, build trading businesses
4. **Endgame** - Defeat the Ender Dragon, explore the Nether safely
