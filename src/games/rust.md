## About Our Rust Server

Welcome to Sheltered Gaming's Rust server! This is where hardcore survival meets intense PvP action. Whether you're a veteran or new to Rust, you'll find challenges and opportunities for epic gameplay.

### Server Rules

- **No hacking or cheating** - We use anti-cheat systems and active moderation
- **Respect other players** - Toxicity and harassment are not tolerated
- **No malicious raiding** - Raid responsibly, don't destroy everything
- **Keep the chat clean** - No spam, racist, or offensive language

1. **Install Rust** on Steam if you haven't already
2. **Copy the server IP** from the Server IP card below
